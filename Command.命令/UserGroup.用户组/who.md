# who

查看当前登录主机的用户终端信息

```sh
[root@localhost Desktop]# who
root :0     2019-11-1622:14(:0)
root pts/0  2019-11-1622:18(:0)
root pts/1  2020-02-1619:22(:0)
root pts/2  2020-02-1619:22(:0)
```

```
root :0     2019-11-1622:14(:0)
```
这一行中 root 后的 ：0 表示这是一个图形界面终端




```
root pts/0  2019-11-1622:18(:0)
```
这一行中 root 后的 pts/0 表示这是一个字符界面的第0个终端

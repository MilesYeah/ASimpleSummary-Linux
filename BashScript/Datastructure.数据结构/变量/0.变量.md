# 变量
定义变量时，变量名不加美元符号$，如：your_name="Miles"

注意，变量名和等号之间不能有空格，这可能和你熟悉的所有编程语言都不一样。同时，变量名的命名须遵循如下规则：
- 首个字符必须为字母（a-z，A-Z）。
- 中间不能有空格，可以使用下划线（_）。
- 不能使用标点符号。
- 不能使用bash里的关键字（可用help命令查看保留关键字）。

变量类型
- 本地变量: `set VARNAME=VALUE`，作用域为整个bash进程； 
- 局部变量: `local VARNAME=VALUE`，作用域为当前代码段；
- 环境变量: `export VARNAME=VALUE`，作用域为当前shell进程及其子进程；
- 位置变量: `$0, $1, $2, ...`，其中`$0` 表示命令本身，`$1`表示第一个参数，依次类推`$3`表示第三个参数
